# Портфолио — Маркаров Михаил Оганесович

Сайт-портфолио специалиста по информационной безопасности.

## Запуск

1. Клонируйте репозиторий или откройте папку `portfolio`.
2. Запустите локальный сервер (например, через расширение Live Server в VS Code или):
   ```bash
   npx serve .
   ```
   Либо откройте `index.html` напрямую в браузере (часть функций может ограничиваться при file://).
3. Для списка документов на странице «Документы»:
   - Поместите PDF в папку `documents/`.
   - Выполните: `node update-documents.js`
   - Обновите страницу в браузере.

## Настройка EmailJS (опционально)

Форма обратной связи по умолчанию использует **mailto** (открытие почтового клиента). Чтобы отправлять письма через сервер:

1. Зарегистрируйтесь на [EmailJS](https://www.emailjs.com/).
2. Создайте сервис, шаблон и получите Public Key.
3. Добавьте на форму (если она есть в разметке) data-атрибуты:
   - `data-emailjs-service` — ID сервиса
   - `data-emailjs-template` — ID шаблона
   - `data-emailjs-public-key` — Public Key
4. Подробнее см. **EMAILJS_SETUP.md**.

Без настроенного EmailJS портфолио работает только через mailto.

## Документы

## Favicon

В проекте используется `favicon.svg` (поддерживается современными браузерами). Для старых браузеров можно сгенерировать `favicon.ico` (32×32) из SVG, например через [favicon.io](https://favicon.io) или аналогичный сервис, и добавить в корень: `favicon.ico`.
