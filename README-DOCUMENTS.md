# Инструкция по добавлению PDF документов

## Автоматическое отображение PDF файлов

### Способ 1: Использование скрипта (Рекомендуется)

1. **Добавьте PDF файлы** в папку `documents/`
   - Например: `documents/certificate1.pdf`, `documents/resume.pdf`

2. **Запустите скрипт обновления:**
   ```bash
   node update-documents.js
   ```

3. **Обновите страницу** в браузере - документы появятся автоматически!

### Способ 2: Ручное обновление JSON

1. Откройте файл `documents-list.json`
2. Добавьте информацию о ваших PDF файлах:
   ```json
   {
     "documents": [
       {
         "filename": "certificate1.pdf",
         "name": "Сертификат 1"
       },
       {
         "filename": "resume.pdf",
         "name": "Резюме"
       }
     ]
   }
   ```
3. Сохраните файл и обновите страницу

### Структура папок

```
portfolio/
├── documents/          ← Поместите PDF файлы сюда
│   ├── certificate1.pdf
│   ├── resume.pdf
│   └── ...
├── documents-list.json ← Автоматически обновляется скриптом
├── documents.html      ← Страница с документами
└── update-documents.js ← Скрипт для обновления списка
```

### Примечания

- Имена файлов должны быть на английском (без пробелов, используйте `_` или `-`)
- Файлы автоматически отображаются на странице `documents.html`
- Каждый PDF отображается в отдельной карточке с предпросмотром
- Пользователи могут открыть PDF в новой вкладке для просмотра или скачивания
